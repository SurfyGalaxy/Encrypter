# Encryption

$\color{red}{\text{THIS IS NOT SECURE. DO NOT USE THIS FOR ANY DATA YOU CARE ABOUT KEEPING PRIVATE}}$

## What it is:
- Diffie-Hellman Key Exchange
- SHA-256 Hashing
- AES-256 Encryption / Decryption
- No imports (excluding `tkinter`)

## Why it's not secure:
<small>This is just a small section of all the vulnerabilities this has</small>
- No salt
- No HMAC
- Vulnerable to timing attacks

## Usage:
1. Download it from <a href="https://github.com/SurfyGalaxy/Encrypter/releases">the releases tab</a>
2. Double click it
3. Navigate the menus by clicking the buttons (It's pretty self explanatory)
